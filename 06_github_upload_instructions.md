## Repository description

AWS and Azure cloud engineering portfolio focused on IAM, monitoring, backup, disaster recovery, incident response and infrastructure support.

## What to upload

Upload:
- README files
- Diagrams
- Runbooks
- Sanitised screenshots
- Test logs

Do not upload:
- Access keys
- Account IDs
- Public IPs if unnecessary
- Real email addresses
- Billing data
- Secrets
- Private documents
- Anything containing personal or confidential information

## GitHub README opening paragraph

This repository documents a self-directed cloud engineering portfolio focused on AWS and Azure infrastructure, IAM security, monitoring, incident response, backup and disaster recovery. The projects are designed to demonstrate practical skills relevant to Junior Cloud Engineer, Cloud Support Engineer, Cloud Infrastructure Support and Cloud Operations roles.
